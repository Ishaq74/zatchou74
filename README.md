# DevStarter Kit 🎨 (English and French version)

## Objective 🎯

Replace WordPress! 😆 A frontend/backend site with mastered and easily maintainable HTML and CSS.
Already optimized for SEO and accessibility.
Allows focusing on branding, design, and content.

### Desired Features and Components

- 📚 Catalog of components (X)
- 🎨 Constants file linking site identity and graphic choices (X)
- 🖼️ Image handling from constants (X)
- 🛠️ Working CMS (X)
- 📬 Functional forms (X)
- 📄 JSON-LD schema integration (X)
- 📑 Pagination (X)
- 🧭 Customizable breadcrumb (X)
- 💬 Comment system (X)
- 👍 Like system (X)
- ⭐ Review system (X)
- 🎨 Design components and animations (X)
- 🔐 Custom authentication (email/password) (X)
- 🛒 E-commerce integration (probably with Next.js) (X)
- 📈 Advanced analytics and reporting (X)
- 🤖 AI-driven recommendations and personalizations (X)
- 🌍 Multi-language support and internationalization (X)
- 🔌 Integration with third-party services and APIs (X)

### Types of Pages, Schemas, and Collections

- **Home Page** OK
- **About Page** OK
- **Services Page** OK
- **Blog Articles** OK
- **Product Pages** (X)
- **FAQ Page** (X)
- **Comment Section** (X)
- **Reviews and Ratings** (X)

## Tech Stack 🛠️

- **Frontend**: [Astro](https://astro.build/)
- **Backend**: Node.js, Netlify Functions
- **CMS**: Decap CMS (formerly Netlify CMS)
- **Authentication**: Netlify Identity (for now)
- **Deployment**: Netlify
- **Styles**: CSS, with tutorials from [Kevin Powell](https://www.youtube.com/kevinpowell)

## Utility 🌟

Starter site kit for developers and non-developers, reconciling frontend and backend. The concept is simple:
- 📚 A catalog of components.
- 🎨 A constants file linking the site's identity and graphical choices for maximum customization.
- 🚀 Basic and advanced features for websites.
- 📄 Collections of all types also serving JSON-LD schemas, content, and head tags.

## Three Act Structure 🎭

### Act 1: Foundations 🔨
- Fix current errors.
- Add basic features like pagination, customizable breadcrumb, comments, likes, reviews, etc.
- Set up components and design animations.
- Use CMS for initial content management.

### Act 2: Expansion 🚀
- Integrate e-commerce, probably with Next.js.
- Customize authentication with email/password.
- Develop a custom CMS.

### Act 3: Perfection 🏆
- Refine all features.
- Optimize performance and SEO.
- Ensure site maintainability and scalability.

### Act 4: Future Ideas 🌟
- Add advanced analytics and reporting.
- Implement AI-driven recommendations and personalizations.
- Expand support for multiple languages and internationalization.
- Integrate with additional third-party services and APIs.
